
  # Let Em Shine

  Shine by Lucy marketing site. Design system: [ShineByLucy — Shine Foundations](https://www.figma.com/design/PGLgfeZHhkOaKXpDOqxdEa/ShineByLucy?node-id=214-2). Legacy Make export: https://www.figma.com/design/tobcsHXyYeYhMt3pmUxu8h/Let-Em-Shine.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
